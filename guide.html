Web Marker

一款轻量级浏览器油猴脚本，让你在本地 HTML 文件上自由高亮、标注文字，刷新不丢失。

v0.1.0 · 油猴脚本 · 零依赖

功能简介

🖍

文字高亮

选中文字，一键添加彩色背景高亮，支持 5 种预设颜色。

🎨

文字变色

选中文字设置字体颜色，5 种醒目色彩让重点一目了然。

🧹

精准擦除

橡皮擦模式下点击高亮区域即可单独删除，或一键清除全部。

🔄

刷新保持

标注数据持久化存储，关闭或刷新页面后重新打开自动恢复。

可用颜色

工具可选颜色
背景高亮 黄 绿 粉 蓝 橙
文字颜色 红 蓝 绿 紫 深橙

安装步骤

1

安装用户脚本管理器

在你的浏览器中安装以下任意一个扩展：

  • 暴力猴（Violentmonkey） — 推荐，开源免费
  • Tampermonkey — 功能最全，使用广泛

支持的浏览器：Chrome、Edge、Firefox、Brave 等主流浏览器。

2

开启「允许访问文件 URL」权限

这一步至关重要！ 不开启此权限，脚本在本地 HTML 文件上完全不会加载。

操作方法因浏览器而异：

Edge 浏览器

  1. 地址栏输入 edge://extensions/ 并回车
  2. 找到你安装的脚本管理器（暴力猴或 Tampermonkey）
  3. 点击 「详细信息」
  4. 找到 「允许访问文件 URL」，将开关 打开

Chrome / Brave 浏览器

  1. 地址栏输入 chrome://extensions/ 并回车
  2. 找到脚本管理器，点击 「详细信息」
  3. 找到 「允许访问文件网址」，将开关 打开

Firefox 浏览器

Firefox 默认允许扩展访问 file:// 页面，通常无需额外设置。若不生效，请在 about:config 中确认 extensions.webextensions.restrictedDomains 未包含 file 协议。

3

安装 Web Marker 脚本

有两种安装方式：

方式 A：手动创建

  1. 点击浏览器工具栏上的脚本管理器图标
  2. 选择 「创建新脚本」（或「新建脚本」）
  3. 清空编辑器中的默认内容
  4. 将 web-marker.user.js 的完整代码粘贴进去
  5. 按 Ctrl + S 保存

方式 B：文件安装

直接将 web-marker.user.js 文件拖入浏览器窗口，脚本管理器会自动弹出安装确认。

4

验证安装

  1. 使用浏览器打开任意本地 HTML 文件（file:/// 开头的地址）
  2. 页面右侧中上方应出现一个 深色小圆形按钮（画笔图标）
  3. 点击该按钮即可展开工具面板
看到按钮了？ 恭喜，安装成功！继续阅读下方使用教程。
没看到按钮？ 请检查：
• 脚本管理器中脚本是否 已启用（开关打开）
• 第 2 步的「允许访问文件 URL」是否已开启
• 是否 重新加载 了页面（修改权限后需要刷新）
• 按 F12 打开控制台查看是否有报错信息

使用教程

基本操作流程

  1. 打开工具面板 — 点击页面右侧的圆形画笔按钮
  2. 选择工具 — 点击「高亮」或「字色」图标激活
  3. 选择颜色 — 在展开的色板中点选一个颜色（默认选中第一个）
  4. 选中文字 — 在页面上用鼠标拖选你要标注的文字，松开鼠标即自动生效
  5. 继续标注 — 工具保持激活状态，可以连续选中不同文字进行标注
  6. 收起面板 — 再次点击圆形按钮折叠工具面板

删除标注

  • 删除单条：点击「擦除」工具图标，然后点击页面上你想删除的高亮文字（激活时高亮区域会显示红色虚线边框提示）
  • 删除全部：点击「清除」工具图标（垃圾桶），一键移除当前页面所有标注

切换颜色

在工具已激活时，直接点击色板上的其他颜色即可切换。后续选中的文字将使用新颜色。已标注的内容不会受影响。

持久化与恢复

标注数据自动保存在浏览器的脚本管理器存储中。当你：

  • 刷新页面 — 所有标注自动恢复
  • 关闭再重新打开 — 同一个文件路径的标注自动恢复
  • 修改了 HTML 文件内容 — 脚本会尝试通过文本匹配恢复，若文字位置变化过大则该条标注不会恢复

常见问题

Q: 在线网页上能用吗？

当前版本（v0.1.0）仅支持本地 file:// 文件。在线网页支持将在后续版本中加入。

Q: 标注数据存在哪里？

存储在脚本管理器（暴力猴/Tampermonkey）的内部存储中，以页面完整 URL 为 key。数据不会上传到任何服务器，完全本地保存。

Q: 换一台电脑能同步标注吗？

目前不支持跨设备同步。如果你的脚本管理器配置了云同步（如暴力猴支持 Dropbox/OneDrive 同步），脚本本身会同步，但标注数据不会。

Q: 高亮后页面显示异常怎么办？

点击工具面板中的「清除」按钮即可一键移除所有标注，恢复页面原始状态。

Q: 支持哪些浏览器？

支持所有能安装暴力猴或 Tampermonkey 的浏览器：Chrome、Edge、Firefox、Brave、Opera 等。

技术说明

  • 零外部依赖：纯原生 JavaScript，无需任何第三方库
  • UI 隔离：工具面板使用 Shadow DOM 渲染，不影响也不被宿主页面的样式影响
  • 智能锚定：通过 XPath + 字符偏移 + 文本上下文三重校验定位标注位置，刷新后精准恢复
  • 跨节点支持：选中横跨多个 HTML 元素的文字也能正确高亮
