# Web Marker - 网页高亮标注

把任意网页变成可高亮、可标注、可演示的交互画布。

## 功能

- **文字高亮**：选中文字添加彩色背景（黄/绿/粉/蓝/橙）
- **文字变色**：选中文字设置字体颜色（红/蓝/绿/紫/深橙）
- **精准擦除**：橡皮擦模式点击删除单条标注，或一键清除全部
- **刷新保持**：标注数据持久化存储，刷新/重开自动恢复
- **自定义光标**：高亮时显示彩色刷子、字色时显示鹅毛笔、擦除时显示橡皮擦，颜色随选中色动态变化
- **快捷键**：`ESC` 一键失活当前工具，快速回到浏览状态
- **UI 隔离**：工具面板使用 Shadow DOM，不受宿主页面样式影响
- **跨节点高亮**：选中横跨多个 HTML 元素的文字也能正确标注

## 当前版本

**v0.1.1** — 油猴脚本（Tampermonkey / Violentmonkey），支持本地 `file://` HTML 文件。

## 快速开始

1. 安装 [暴力猴](https://violentmonkey.github.io/) 或 [Tampermonkey](https://www.tampermonkey.net/)
2. 在扩展管理页面开启 **「允许访问文件 URL」**
3. 在脚本管理器中新建脚本，粘贴 `web-marker.user.js` 内容并保存
4. 打开任意本地 HTML 文件，右侧出现工具按钮即安装成功

详细安装说明请参阅 [guide.html](guide.html)。

## 使用方法

1. 点击页面右侧圆形按钮展开工具面板
2. 选择工具（高亮/字色）并选择颜色
3. 在页面上拖选文字，松开鼠标自动生效
4. 擦除：选择橡皮擦工具后点击高亮区域删除

## 技术栈

- 纯原生 JavaScript，零外部依赖
- Shadow DOM 实现 UI 隔离
- XPath + 字符偏移 + 文本上下文三重锚定
- GM_setValue / GM_getValue 持久化存储
